# Family-Expenses-management-servicenow
A ServiceNow-based application to manage family expenses using workflows, approvals, and dashboards for better financial tracking and insights.
# 💰 Family Finance Management using ServiceNow

## 📌 Project Overview
This project demonstrates how ServiceNow can be used beyond IT Service Management to solve real-world problems like family expense tracking.

It enables users to record, categorize, and monitor household expenses using ServiceNow workflows, dashboards, and automation features. The system helps improve financial awareness and control spending effectively.

---

## ✨ Key Features

- 📝 **Expense Submission Form**
  - Users can add expenses such as groceries, rent, education, etc.

- 🗂 **Automatic Categorization**
  - Expenses are grouped into categories like Food, Utilities, Transport, and Education.

- ✅ **Approval Workflow**
  - Expenses go through an approval process for better financial control.

- 📊 **Interactive Dashboard**
  - Visual representation of spending trends (monthly & yearly).

- 🔔 **Notifications & Alerts**
  - Alerts when expenses exceed defined limits.

---

## 🛠 Technologies Used

- **Platform:** ServiceNow
- **Scripting:** JavaScript, Glide API
- **Automation:** Business Rules, Flow Designer, Scheduled Jobs
- **UI:** Service Portal

---

## 🚀 How to Run the Project

1. Import the Update Set into your ServiceNow instance.
2. Open Service Portal.
3. Navigate to **Expense Submission**.
4. Add sample expenses.
5. View reports in the dashboard.
6. Test approval workflow and notifications.
